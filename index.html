Public Flashcards Repository

GET request for API using CURL. example

curl --location 'https://aliabbasgabol.pythonanywhere.com/Flashcards'

POST request for api using CURL. example

curl --location 'https://aliabbasgabol.pythonanywhere.com/Flashcards' \
--header 'Content-Type: application/json' \
--data '{
    "question": "what is fire",
    "answer":"fire"
}'

PUT request for api using CURL. example


curl --location --request PUT 'https://aliabbasgabol.pythonanywhere.com/Flashcards' \
--header 'Content-Type: application/json' \
--data '{
    "question": "what is fire",
    "answer":"fire",
    "id": 1
}'

DELETE request for api using CURL. example

 
  curl --location --request DELETE 'https://aliabbasgabol.pythonanywhere.com/Flashcards' \
--header 'Content-Type: application/json' \
--data '{
    "id": 1
}'
